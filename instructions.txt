ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ

1. ОБЩАЯ ЛОГИКА
   Программа выполняет сортировку данных по выбранному критерию - 
   типу поля в блоке исходного файла и направлению сортировки.
   Для работы используется алгоритм внешнего сканирования, что позволяет
   обрабатывать файлы любого объема без переполнения оперативной памяти.

2. ДАТЫ И ГРУППЫ
   Сравнение дат производится через преобразование в формат ГГГГММДД.
   Значения 00000000 и 99999999 являются техническими якорями для 
   корректного позиционирования пустых или крайних значений в списке.

3. УПРАВЛЕНИЕ
   - Выбор в меню: цифровые клавиши 1-5.
   - Выход/Отмена: клавиша ESC (работает в меню и при вводе текста).
   - Подтверждение: ENTER.

4. РАБОТА С ФАЙЛАМИ
   - Вводите только имя файла без расширения (.txt добавится само).
   - Если файл с таким именем существует, программа автоматически 
     создаст копию с индексом, например: имя(1).txt.
